# My-CV
## README: Portfolio CV Website

### 1. Project Info

This project is a responsive personal portfolio and digital CV designed for **Lumia Qureshi**. It features a modern 3D aesthetic with a "Design. Debug. Deploy." hero section, interactive project cards, and smooth scroll animations. The layout is optimized for both desktop and mobile viewing, utilizing a sophisticated color palette of soft creams, blacks, and accent tones.


### 2. Technologies Used

The project is built using a clean, "no-framework" approach to ensure maximum speed and easy customization:

* **HTML5**: For semantic structure and content organization.
* **CSS3**: Featuring custom properties (variables), Flexbox for layout, and `@keyframes` for the "fade-in" and "slide-in" animations.

### 5. Project Features

* **Interactive UI**: Cards that "jump" on hover to engage users.
* **Responsive Design**: A fluid layout that stacks content vertically on mobile devices for better readability.
* **Skill Highlight**: A "pill-style" skill section categorized for quick scanning by recruiters.
* **Contact Integration**: Direct links to Email, LinkedIn, and GitHub with custom 3D hover effects.
